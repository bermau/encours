version v2 
BUT : convernir le fichier des correspondants en un fichier exploitable sur un tableur.

  - procéder à une dictionnaire des correspondant depuis Synergy V5. Exporter les données dans une imprimante virutelle (exemple la 58).
  - par ftp importer le fichier fi58
  - renommer ce fichier en corr dans le répertoire présent.
  - lancer ce programme par IDLE ou tout autre procédé avec python >3.0
  - le programme crée un fichier csv
